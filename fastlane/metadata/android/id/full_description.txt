<p style="text-align: left;">ZipXtract adalah aplikasi Android open-source yang dirancang untuk menangani berbagai format arsip, termasuk:</p>
<ul>
<li>RAR (dengan dukungan RAR5 dan multi-volume)</li>
<li>ZIP (termasuk arsip terproteksi kata sandi dan split)</li>
<li>TAR</li>
<li>BZ2</li>
<li>GZ</li>
<li>7z (dengan proteksi kata sandi dan dukungan multi-volume)</li>
<li>XZ</li>
<li>JAR</li>
<li>...</li>
</ul>

<h2>Fitur</h2>

<h3>Ekstraksi:</h3>
<ul>
<li>Mendukung berbagai format: RAR, ZIP, TAR, BZ2, GZ, 7z, XZ, JAR, dll.</li>
<li>Menangani file ZIP dan 7z terenkripsi.</li>
<li>Mengekstrak arsip RAR multi-volume.</li>
<li>Mengekstrak arsip multi-volume.</li>
<li>Mengekstrak file ZIP split.</li>
</ul>

<h3>Pembuatan:</h3>
<ul>
<li>Membuat arsip ZIP dan 7z, dengan opsi enkripsi.</li>
<li>Membuat arsip ZIP split.</li>
</ul>

<blockquote>
<p><strong>CATATAN:</strong><br />
Versi F-Droid tidak menyertakan fitur ekstraksi RAR dan kompresi 7z karena pembatasan lisensi. Untuk versi lengkap, silakan gunakan rilis dari GitHub atau IzzyOnDroid.</p>
</blockquote>

<h2>Tips Penggunaan</h2>
<p>Tips umum untuk ekstraksi dan pembuatan arsip:</p>
<ul>
<li>Tekan lama file dan folder atau klik ikon file/folder untuk memilihnya untuk diarsipkan.</li>
<li>Klik file untuk menampilkan menu ekstraksi dan memilih opsi ekstraksi.</li>
<li>File yang diekstrak menggunakan menu "Buka Dengan" dari OS disimpan di <strong>Penyimpanan Internal/ZipXtract</strong> secara default. Ubah di pengaturan aplikasi.</li>
<li>Lokasi default untuk mengarsipkan atau mengekstrak arsip (dalam aplikasi) adalah direktori induk dari file tersebut. Ini dapat diubah di pengaturan aplikasi.</li>
</ul>
